Okay, so here's a simple little web page. Note how I do almost all my styling in a separate .css file, to keep my html clean, but almost all my css selectors refer to classes, not general elements. This is so my styles don't clash with one another--especially important if you're working in teams.

Nonetheless, note the styling I used to center the image--how I had to set display to block for the margin to take effect.

Other things to note inside the .css file:

  • I use the background-image attribute, along with a few useful relatives, to create the jumbotron (the cat-explosion picture thingy).
  • Using the :hover modifier, I have added an opacity property to the jumbotron that changes when the user hovers the cursor over the image.
